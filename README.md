# SwiftCryptoMiner
It's a concept of cryptocurrency miner based on Swift.

It's just a simple example of cryptocurrency mining, based on Swift. In future it might be updated for Metal support.

Algorithm's support:
- [x] SHA-256d
- [ ] ethash
- [ ] CryptoNight
